## suspend 함수
***
일시 중단 가능한 함수로, 함수 내 일시 중단이 가능한 작업이 있다는 것을 말한다.
- 코루틴 내부, <code style="color: #eb5657;">suspend</code> 함수 내부에서만 호출할 수 있다.
<br/>

```kotlin
import kotlinx.coroutines.*

fun main() = runBlocking {
    test1()
    test2()
}

suspend fun test1() {
    delay(3000)
    println("Test1")
}

suspend fun test2() {
    delay(2000)
    println("Test2")
}


// Test1
// Test2
```
