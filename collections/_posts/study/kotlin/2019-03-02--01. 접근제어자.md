## Kotlin 접근 제어자
<code style="color: #eb5657;">private</code>
- 외부에서 접근 불가능
<br/>

<code style="color: #eb5657;">public</code>
- 모든 곳에서 접근 가능
- default 접근 제어자
<br/>

<code style="color: #eb5657;">internal</code>
- 같은 모듈 내에서만 접근 가능
<br/>

<code style="color: #eb5657;">protected</code>
- 같은 패키지내에서, 혹은 상속 받은 클래스에서 접근 가능
<br/>
<br/>
